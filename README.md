Git Like A Boss
===============


1. clone the repo

	git clone git@github.com:pj4533/setupfiles.git

2. Install zsh

	chsh -s /bin/zsh

3. Copy zshrc (or symlink)

	cp .zshrc ~/.zshrc

4. Copy gitconfig

	cp .gitconfig ~/.gitconfig

5. Change name/email in ~/.gitconfig
 
	mate ~/.gitconfig

6. Restart terminal

7. Rock out
